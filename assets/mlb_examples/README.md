# MLB Example Images

This directory contains example images from MLB pitchers to demonstrate proper technique for each metric.

## Image Naming Convention

Images should be named according to the metric key:
- `head_behind_hip.jpg` (or `.jpeg`, `.png`) - For "Head Behind Hip Toward Plate" metric
- `stride_length.jpg` (or `.jpeg`, `.png`) - For "Stride Length" metric
- `chest_closed_ffp_deg.jpg` (or `.jpeg`, `.png`) - For "Chest Closed at Foot Strike" metric
- `throw_elbow_vs_shoulder_line_ffp.jpg` (or `.jpeg`, `.png`) - For "Elbow Height at Foot Strike" metric
- `elbow_bend_ffp_deg.jpg` (or `.jpeg`, `.png`) - For "Arm Angle at Foot Strike" metric
- `upper_body_lean_release.jpg` (or `.jpeg`, `.png`) - For "Upper-Body Lean at Release" metric

## Adding Images

1. Place your image file in this directory with the appropriate name
2. Supported formats: `.jpg`, `.jpeg`, `.png`
3. The image will automatically be displayed when viewing the corresponding metric in the results page
